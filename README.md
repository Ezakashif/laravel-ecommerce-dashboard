# Laravel Ecommerce Admin Dashboard

An advanced ecommerce admin dashboard built with Laravel for managing products, orders, categories, and users.

## Features

- Product Management
- Category Management
- Order Management
- User Management
- Dashboard Analytics
- Responsive Admin UI
- Authentication System

## Tech Stack

- Laravel
- PHP
- MySQL
- SCSS
- JavaScript

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
